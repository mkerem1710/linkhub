# linkhub

Open-source Linktree alternative.  
Design it in 1 minute → download your code with one click → deploy free forever on GitHub Pages.

Live demo → https://mkerem1710.github.io/linkhub

- 8 beautiful themes  
- Avatar + bio + unlimited links  
- Drag & drop ordering  
- Download clean, standalone HTML in one click  
- No signup, no payment, no tracking  

The code is 100% yours.  
Fork it, modify it, host it anywhere.
